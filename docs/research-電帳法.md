# 電子帳簿保存法（電帳法）調査レポート

## 調査日

2026-03-30

## 背景

請求書・納品書をPDF生成してメール（SendGrid）で得意先に送信するシステムを開発中。
送信側（東海鋼材工業様）としての法的義務を整理する。

## 前提

- **東海鋼材工業様の売上高は5,000万円超** → 検索要件は必須
- 現在は紙で請求書・納品書を発行しており、紙の控えを保存 → **合法**
- 新システムでメール（PDF）送信に切り替えると「電子取引」となり、保存ルールが変わる
  - 送信したPDFの控えを**紙に印刷して保存するのはNG**
  - **電子データのまま保存する義務**が発生する

## 結論

| 項目 | 結論 |
|------|------|
| メールPDF送信は「電子取引」か | **該当する** |
| 送信側の保存義務 | **ある**（PDFの控えを電子データのまま保存） |
| 電子送信した控えを紙印刷で保存できるか | **不可**（電子データのまま保存が必須。2024年1月完全義務化済） |
| 保存期間 | **7年間**（法人） |
| 検索要件 | **必須**（売上高5,000万円超のため） |

## 検索要件の詳細

以下の検索機能を**システムに実装する必要がある**：

1. **取引年月日** での検索（範囲指定可）
2. **取引金額** での検索（範囲指定可）
3. **取引先名** での検索
4. 上記の**組み合わせ検索**

## 改ざん防止措置（いずれか1つ）

| 方法 | コスト | 内容 |
|------|--------|------|
| 事務処理規程の策定 | **低（推奨）** | 国税庁サンプルを参考に規程を作成・運用 |
| システムで履歴管理 | 中 | 訂正・削除の履歴を残す仕組み |
| タイムスタンプ付与 | 高 | 認定タイムスタンプサービスの利用 |

## 法的根拠

### 条文

| 根拠 | 内容 |
|------|------|
| **電子帳簿保存法 第7条** | 電子取引データの保存義務（本則） |
| **施行規則 第2条第6項第5号** | 検索要件の具体的内容 |
| ├ イ | 取引年月日・取引金額・取引先を検索条件として設定できること |
| ├ ロ | 日付・金額について範囲を指定して検索できること |
| └ ハ | 二以上の任意の記録項目を組み合わせて検索できること |
| **施行規則 第4条第1項 柱書** | 売上高5,000万円以下の場合の検索要件免除（※東海鋼材工業様は該当しない） |

### 国税庁 一問一答（電子取引関係）

| Q番号 | 内容 |
|-------|------|
| 問42 | 検索機能で注意すべき点 |
| 問43 | 組み合わせ検索の範囲（「AかつB」のみ、「A又はB」は不要） |
| 問44 | システムがない場合の検索機能確保方法（ファイル命名規則等） |
| 問45 | 5,000万円以下の判定方法（基準期間＝2期前の売上高） |
| 問51 | 検索要件の「取引金額」は税込・税抜どちらでもよい（統一すること） |

## 罰則・ペナルティ

### 直接的な罰則

電帳法自体には、保存義務違反・検索要件不備・改ざん防止措置不備に対する**直接的な刑事罰（罰金・懲役等）は規定されていない**。

### 間接的な不利益（実質的に大きなダメージ）

| 不利益 | 根拠条文 | 内容 |
|--------|----------|------|
| 青色申告の承認取消 | 法人税法第127条第1項 | 欠損金繰越控除、各種優遇措置が使えなくなる |
| 経費（損金）否認 | 法人税法第22条第3項 | 保存不備の取引の経費が認められない |
| 消費税の仕入税額控除否認 | 消費税法第30条第7項 | 請求書の保存不備で控除できない |
| 推計課税 | 法人税法第131条 | 青色申告取消後、実額より不利な課税の可能性 |

### 重加算税の加重措置（令和4年度税制改正）

電子取引データの**隠蔽・仮装**があった場合、通常の重加算税に**10%が加重**される（国税通則法第68条第4項）。

| 区分 | 通常税率 | 加重後 |
|------|----------|--------|
| 過少申告 | 35% | **45%** |
| 無申告 | 40% | **50%** |

### 根拠条文・参考情報

| 根拠 | 内容 |
|------|------|
| **国税通則法 第68条第4項** | 電子取引データの隠蔽・仮装に対する重加算税10%加重 |
| **法人税法 第127条第1項第1号** | 帳簿書類の保存不備による青色申告承認取消 |
| **法人税法 第22条第3項** | 損金算入の要件（保存不備で否認の根拠） |
| **消費税法 第30条第7項** | 請求書等の保存を仕入税額控除の要件とする規定 |
| **法人税法 第131条** | 青色申告取消後の推計課税 |

### まとめ

罰金・懲役はないが、税務調査で指摘された場合の金銭的ダメージは大きい。「罰則がないから対応不要」とは言えない。

## システムへの影響

### 必須対応

1. **PDF控えの電子保存**: S3に保存（設計済み）→ **7年間保持ポリシーの設定が必要**
2. **検索機能の実装**: 処理済み一覧に日付・金額・取引先の範囲指定・組み合わせ検索を追加
3. **改ざん防止措置**: 事務処理規程の策定（最低限）、またはシステムで訂正・削除履歴を管理

### 要件定義で確認すべき事項

- 改ざん防止措置の方針（規程 or システム対応）
- 既存の電帳法対応状況（他システムで対応済みか）
- 保存期間の方針（7年 or 安全策で10年）
