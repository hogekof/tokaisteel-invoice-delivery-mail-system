**有効期限**: 2026年2月28日

---

## 1. システム概要

軽技Web（BIツール）から売上明細データを取得し、営業担当者による確認・確定を経て、請求書・納品書PDFを顧客にメール送信するシステムを開発いたします。

### 主な機能
- 部門選択画面（URLブックマーク対応）
- 軽技Webからのデータ自動取得（常に最新データを表示）
- 確認待ち一覧（請求書/納品書タブ切替、送信ON/OFF設定※、営業担当フィルター）
- 書類詳細表示（明細確認）
- 処理予定一覧（確定解除可能）
- 処理済み一覧（過去1週間分）
- PDF自動生成（請求書/納品書）
- メール自動送信（毎日10時バッチ処理）
- IP制限によるアクセス制御

※送信ON/OFF設定はブラウザに保存され、データ再取得後も保持されます

## 5. 技術構成

### 5.1 アーキテクチャ

```
[ユーザー] → [API Gateway] → [Lambda (Dash)] → [DynamoDB]
                                    ↓               ↓
                              [軽技Web]         [S3 (PDF)]

[EventBridge] → [Lambda (Batch)] → [SendGrid]
   (10:00)
```

### 5.2 データの流れ

```
軽技Web → データ取得 → 確認待ち一覧（画面表示のみ）
                            ↓
                    営業担当者が確定
                            ↓
                    DynamoDBに保存 → PDF生成 → 処理予定
                                                  ↓
                                        毎日10時にメール送信
                                                  ↓
                                              処理済み
```

※確認待ちデータは毎回軽技Webから最新を取得（古いデータによる誤送信を防止）

### 5.3 技術スタック

| レイヤー | 技術 |
|----------|------|
| フロントエンド | Dash (Plotly) 2.x / Python 3.12 |
| バックエンド | Flask (Dash内蔵) |
| データベース | Amazon DynamoDB |
| ファイル保存 | Amazon S3 |
| 軽技Web連携 | Selenium (Headless Chrome) |
| PDF生成 | WeasyPrint |
| メール送信 | SendGrid API v3 |
| インフラ | AWS Lambda, API Gateway, EventBridge |
| デプロイ | AWS SAM |
| アクセス制御 | API Gateway IP制限 |
